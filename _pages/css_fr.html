L'onglet Styles CSS de la boîte d'édition de tout élément Karuta permet d'en modifier l'apparence. La partie du haut permet de spécifier l'apparence du libellé de l'élément et la partie du bas celle de son contenu. Le contenu d'un élément de structure correspond à tous les éléments qu'il contient, alors que le contenu d'un élément ressource correspond à la donnée contenue dans l'élément (texte, date, hyperlien, etc).

Des champs pré-définis permettent d'ajuster le poids, le style, la taille et la couleur de la police, l'alignement du texte, l'espace avant (instruction CSS padding-top) et la couleur du fond. Un champ de texte libre permet aussi d'ajouter du code CSS bien formé (champ Autres instrucions CSS).

Si plusieurs éléments d'un portfolio doivent avoir le même style CSS, il est possible de déclarer une classe CSS dans l'élément Root du portfolio (champ CSS) puis de spécifier le nom de la classe dans chaque élément visé (champ Classe CSS).
